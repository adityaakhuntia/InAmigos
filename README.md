# InAmigos Foundation - NGO Awareness Website

A high-fidelity, editorial-style website designed and developed to spread awareness about the **InAmigos Foundation** and its various social impact initiatives. This project was completed as part of a **Web Developer Internship**.

## 🌟 Project Overview
This website serves as a digital platform to showcase the NGO's mission, ongoing projects, and social impact metrics. It is designed with a premium, "Editorial / Luxury Humanist" aesthetic to establish trust and professional authority, encouraging visitors to join the movement as volunteers or donors.

## 🚀 Role & Contribution
**Position:** Web Developer Intern
- Architected a modern, responsive UI/UX from scratch using a luxury humanist design philosophy.
- Implemented a "Liquid Glass" design system using advanced CSS3 techniques and custom properties.
- Developed dynamic front-end features including scroll-reveal animations, custom cursor interactions, and parallax effects.
- Integrated social impact reporting through branded campaign sections (#InAmigos).

## ✨ Key Features
- **High-Fidelity UI**: Editorial-style typography utilizing *Fraunces* and *DM Sans* for a premium look.
- **Dynamic Interactions**: Custom animated loader, scroll-based section reveals, and interactive project cards with smooth transitions.
- **Project Showcase**: Detailed highlights of core initiatives including Project VIKAS, SEVA, and BACHPANSHALA.
- **Social Impact Trackers**: Live counter animations visualizing real-world contributions (Meals, Trees, Interns).
- **Branded Hashtag Belt**: A continuous scrolling marquee of official campaign tags to drive social media awareness.
- **Fully Responsive**: Optimized for a seamless experience across all screen sizes (Mobile, Tablet, Desktop).

## 🛠️ Technology Stack
- **HTML5**: Semantic structure optimized for SEO and accessibility.
- **CSS3**: Modern layouts (Flexbox/Grid), custom variables, keyframe animations, and glassmorphism effects.
- **Google Fonts**: Integration of premium typography for an editorial feel.
- **FontAwesome**: High-quality vector iconography for intuitive navigation.
